Increment version tag. If the latest version tag is in a format of ints you can increment specific version numbers.

Use [0 | +] chars to set incrementing. 0 will that version number the same. + will increment that version number.

Example: If a latest tag is 1.1.1, and the increment string is set to 0.0.+, then the new incremented tag will be 1.1.2.

If a tag is not matched using the increment feature, the build will fail.

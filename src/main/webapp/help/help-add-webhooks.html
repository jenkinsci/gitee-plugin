Add webhook to Gitee repos of choice. You must have adminstrator access to use this feature.

Make sure your Gitee Connection is set properly, and your root URL is set to something other than localhost or 127.0.0.1.

If you are configuring a project for the first time and it is not working, make sure to save the project with your Gitee Connection saved, or you will run into errors.

When you have set up the configuration, hit the Connect Webhook button. The request will succeed if the webhook is created, or the webhook exists already.
